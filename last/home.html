This innovative system integrates IOT sensors, web interfaces, and Java-based software to provide efficient and user-friendly parking management capabilities.

Objective:

Collecting slot data using sensors. Interpreting data using MCU board. Sending data to server using internet protocols. Dynamically reflecting the status onto the webpage.
